# Ivy Chebet Portfolio

A premium, Apple-inspired personal portfolio built with HTML5, CSS3, and vanilla JavaScript.

## Structure

- `index.html` - Semantic page markup and SEO metadata
- `style.css` - Responsive visual system, glassmorphism, animations, and dark mode
- `script.js` - Typing animation, Intersection Observer reveals, nav, theme toggle, and card tilt
- `assets/images` - Hero and project visuals
- `assets/icons` - Favicon
- `assets/cv` - Downloadable CV placeholder

## Run

Open `index.html` in a browser, or serve the folder with any static file server.
