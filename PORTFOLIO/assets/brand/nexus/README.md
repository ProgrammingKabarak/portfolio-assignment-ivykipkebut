# Nexus Logo Kit

Premium logo assets for Nexus, a personal analytics platform for productivity, finances, learning, habits, and growth.

## Files

- `nexus-logo-horizontal.svg` - horizontal brand logo
- `nexus-logo-horizontal.png` - transparent horizontal PNG
- `nexus-icon.svg` - standalone icon
- `nexus-icon.png` - transparent icon PNG
- `nexus-favicon.svg` - favicon vector
- `nexus-favicon.png` - favicon PNG

## Palette

- Indigo: `#6366F1`
- Emerald: `#10B981`
- Dark Gray: `#111827`
- White: `#FFFFFF`
