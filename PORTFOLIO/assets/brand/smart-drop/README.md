# Smart Drop Logo Kit

Premium vector logo assets for Smart Drop, an IoT-powered solar irrigation monitoring system.

## Files

- `smart-drop-logo-full.svg` - full-color horizontal logo
- `smart-drop-logo-monochrome.svg` - single-color horizontal logo
- `smart-drop-mark.svg` - standalone transparent vector mark
- `smart-drop-app-icon.svg` - rounded app icon vector
- `smart-drop-favicon.svg` - favicon vector
- `smart-drop-logo-full.png` - transparent full-color PNG
- `smart-drop-logo-monochrome.png` - transparent monochrome PNG
- `smart-drop-mark.png` - transparent logo mark PNG
- `smart-drop-app-icon.png` - app icon PNG
- `smart-drop-favicon.png` - favicon PNG

## Palette

- Green: `#22C55E`
- Dark Green: `#16A34A`
- Blue: `#0EA5E9`
- White: `#FFFFFF`
- Ink for monochrome: `#111827`
