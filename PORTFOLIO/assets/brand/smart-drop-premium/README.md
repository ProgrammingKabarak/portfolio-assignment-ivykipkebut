# Smart Drop Premium Logo Kit

Award-oriented brand system for Smart Drop, an intelligent solar-powered irrigation monitoring system.

## Deliverables

- `smart-drop-primary.svg` / `.png` - primary horizontal logo
- `smart-drop-stacked.svg` / `.png` - stacked logo
- `smart-drop-icon.svg` / `.png` - icon only
- `smart-drop-app-icon.svg` / `.png` - app icon
- `smart-drop-favicon.svg` / `.png` - favicon
- `smart-drop-light.svg` / `.png` - light version
- `smart-drop-dark.svg` / `.png` - dark mode version
- `smart-drop-monochrome.svg` / `.png` - monochrome version
- `brand-guidelines.svg` / `.png` - brand guidelines preview

## Brand Message

Smart Drop empowers farmers with intelligent irrigation through solar-powered IoT technology, helping conserve water, improve crop productivity, and support sustainable agriculture.

## Palette

- Primary Green: `#22C55E`
- Forest Green: `#15803D`
- Sky Blue: `#0EA5E9`
- Dark Slate: `#111827`
- White: `#FFFFFF`

## Usage

Use the primary logo for websites, presentations, GitHub, and product documents. Use the icon for small spaces, mobile apps, favicon use, and packaging marks. Use the dark version on dark UI surfaces and the monochrome version when color reproduction is limited.
