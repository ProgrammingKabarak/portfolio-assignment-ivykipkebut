# TrustPay Logo Kit

Premium fintech logo assets for TrustPay, a secure QR payment platform.

## Files

- `trustpay-logo.svg` - brand logo
- `trustpay-logo.png` - transparent brand PNG
- `trustpay-logo-dark.svg` - dark mode logo
- `trustpay-logo-dark.png` - dark mode PNG
- `trustpay-icon.svg` - standalone icon
- `trustpay-icon.png` - transparent icon PNG
- `trustpay-favicon.svg` - favicon vector
- `trustpay-favicon.png` - favicon PNG

## Palette

- Blue: `#2563EB`
- Navy: `#1E3A8A`
- Green: `#10B981`
- White: `#FFFFFF`
